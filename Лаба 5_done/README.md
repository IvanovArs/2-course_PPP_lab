Данная программа представляет собой десктопное приложение для управления базой данных клиентов интернет-провайдера. Программа предоставляет графический интерфейс для 
базовых операций с клиентами (создание, изменение данных, удаление), а также просмотр всех данных в таблице, назначения скидок и подсчета статистики (общая выручка, 
количество клиентов, средняя стоимость услуг). В программе используется база данных SQLite, вследствие чего вся информация хранится в одном файле, а также это 
обеспечивает надежное согласование с приложением без необходимости установки отдельного сервера БД.
