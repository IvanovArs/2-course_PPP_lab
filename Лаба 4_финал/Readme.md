Основные функционал и реализованные задачи:
1)	Минимально 2 окна.
2)	Перегрузка конструктора, чтобы при разных сценариях работы изменялись визуальные отображения на форме.
3)	Выгрузка из файла и сохранение в файл данных.
4)	Добавление, изменение и удаление данных, ввод данных вручную. Изменения, связанные с данными тремя функциями должны визуально отображаться.
5)	Сортировка данных.
6)	Проверку полей правильности ввода. 
7)	Функционал консольного приложения.
8)	Проверки полей и логики работы программы.
11)	Компиляция программы в исполняемый файл.
